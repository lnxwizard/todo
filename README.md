# todo
A simple to-do list app that you can use from the terminal.

# About
## Usage
```shell
todo [flags]
```

## Examples
```shell
todo --add Go Walk
```
```shell
todo --list
```
```shell
todo --complete 2
```
```shell
todo --delete 5
```

## Available Flags
- `--add`: Add a new to-do.
- `--delete`: Delete a to-do by index.
- `--complete`: Mark a to-do as completed by index.
- `--list`: List all to-dos.
- `-h`, `--help`: Print help message for todo.
