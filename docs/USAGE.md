# Usage of All Commands

## add
Add a new to-do.

```
todo add [your to-do]
```

Example:
```
todo add Take a walk
```

## complete
Mark a to-do as completed by index.

```
todo complete [index]
```

Example:
```
todo complete 1
```

## completion
Generate the autocompletion script for the specified shell.

```
todo completion [command]
```

Example:
```
todo completion bash
```

## delete
Delete a to-do by index.

```
todo delete [index]
```

Example:
```
todo delete 1
```

## list
List all to-do's.

```
todo list
```

## help
Help about any command.

```
todo help [command]
```

Example:
```
todo help add
```
